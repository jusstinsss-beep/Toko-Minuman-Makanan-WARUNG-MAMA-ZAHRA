# User Interface Mockups

This document contains user interface mockups, ASCII diagrams, screen layouts, and design specifications for the following sections of the Point of Sale (POS) system:

## 1. Dashboard
- Overview of sales, customer inquiries, and system notifications  
```
+------------------+
|      Dashboard    |
|------------------|
| Sales Today: $X  |
|----------------------------------|
|  [ Inventory ]  [ Reports ]      |
|----------------------------------|
|  Notifications:                  |
| - Alert 1                        |
| - Alert 2                        |
+------------------+
```

## 2. Inventory
- Manage products, stock levels, and pricing  
```
+------------------+
|    Inventory      |
|------------------|
| Product  | Stock | Price |
|----------|-------|-------|
| Item 1   | 100   | $X    |
| Item 2   | 50    | $Y    |
+------------------+
```  

## 3. Reporting
- View sales reports by date, category, and more  
```
+------------------+
|    Reports       |
|------------------|
| Date       | Sales |  
|------------|-------|
| 2026-03-01 | $X    |
| 2026-03-02 | $Y    |
+------------------+
```

## 4. Design Specifications
- Colors: #2F4F4F (Dark Slate Gray), #FF4500 (Orange Red)
- Fonts: Arial for all text
- Button Styles: Rounded corners, 12px padding, gradient background
  
The layout should keep usability in mind, ensuring users can navigate easily between sections.
